# Traffic Collision Analysis

## Project Overview
This project analyzes NYC traffic collision data to identify high-risk locations, time periods, and contributing factors associated with severe accidents.

## Business Problem
Traffic collisions result in injuries, fatalities, and economic costs. The goal of this analysis was to identify patterns that could support data-driven safety interventions.

## Tools Used
- SQL
- Tableau
- Python

## Key Insights
- Certain boroughs experienced significantly higher collision severity
- Peak accident frequency occurred during commuter hours
- Weather and traffic density impacted injury rates
